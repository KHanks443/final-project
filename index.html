Baseball Training Log

A simple website for tracking workouts and practice sessions. Add a session, then view it in the logs. Built with HTML/CSS/JavaScript + Express + SQLite.

Quick Checklist

  • 3 pages linked with a navbar
  • Mobile-friendly layout and menu
  • Form validated in JavaScript with clear errors
  • Backend API routes + SQLite database

Next Steps

  1. Go to Add Session
  2. Submit the form (POST to backend)
  3. See the data on View Logs (GET from backend)
